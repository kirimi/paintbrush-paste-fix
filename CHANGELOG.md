# Changelog

All notable changes to Paintbrush Paste Fix will be documented in this file.

## 1.0.0 - 2026-07-14

- Added automatic Retina clipboard image normalization for Paintbrush 2.6.
- Added PNG and TIFF clipboard output.
- Added a menu bar status item and manual clipboard fix command.
- Added Universal 2 support for Apple Silicon and Intel Macs.
- Added automated tests for 1× and 2× image representations.
